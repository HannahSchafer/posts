## Install and Run

1. Install dependencies: `npm install`
2. Run the json-server server: `npm run start-server`
3. In another Terminal window, run the React server: `npm start`
4. Navigate to `http://localhost:3000/`

## Run tests

1. `npm test`

## Technologies Used

1. React
2. Typescript
3. Webpack
4. Babel
5. Json-Server
6. Prettier
7. ESlint
8. React-Icons
9. Moment.js
